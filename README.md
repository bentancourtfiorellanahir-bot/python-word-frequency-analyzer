# Python Word Frequency Analyzer

This project analyzes a text file and calculates the frequency of each word using Python and regular expressions.

## Features

- Reads a text file
- Extracts words using regex
- Counts how many times each word appears
- Displays the most common words

## Technologies

- Python
- Regular Expressions (re module)

## How it works

The program reads a text file line by line, extracts words using a regular expression, and stores the frequency of each word in a dictionary. The results are then sorted and the most common words are displayed.

## Example Output

Top words:

the : 56  
python : 34  
data : 22
